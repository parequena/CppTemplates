## Installation and compilation
1. Compile this example with any compiler.
2. In my case, clang++, execute: `clang++ -std=c++20 main.cpp -o simple`
3. Execute code: `./simple`

All in one:

`clang++ -std=c++20 main.cpp -o simple && ./simple`
